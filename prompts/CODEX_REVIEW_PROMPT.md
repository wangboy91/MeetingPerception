# Codex 自检与重构提示词

请基于当前仓库代码和工程文档做一次代码自检。

重点检查：

1. 是否符合 `AGENTS.md` 的开发规范；
2. 是否符合当前阶段任务的验收标准；
3. 是否有模块职责混乱；
4. 是否有业务逻辑写死在 UI；
5. 是否有 Prompt 散落在代码里；
6. 是否有 API Key 或敏感信息进入代码；
7. 是否有日志打印完整会议内容；
8. WebSocket 消息 schema 是否统一；
9. ASR/VAD/LLM 是否有 Provider 抽象；
10. 当前代码是否能一条命令启动。

请输出：

- 发现的问题；
- 优先级；
- 建议修改方案；
- 可以直接执行的修复步骤；
- 修改后的测试方式。

如果问题明确，请直接修复。
